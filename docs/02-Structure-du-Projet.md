# Structure du Projet

```
easy-group/
├── base-back/                  # Backend Java Spring Boot
├── base-front/                 # Frontend Angular
├── mysql/                      # Fichiers d'initialisation MySQL
├── docker-compose.yml          # Config par défaut
├── docker-compose.prod.yml     # Config production
├── docker-compose.staging.yml # Config staging
├── .env                        # Variables partagées
├── .env.prod                   # Variables production
├── .env.staging                # Variables staging
├── .github/workflows/          # Fichiers GitHub Actions
```
